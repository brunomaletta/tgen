\doxysubsubsection{Detailed Description}
Defines a set of sequences, subject to restrictions. 

A uniformly random {\ttfamily \doxylink{structtgen_1_1sequence_1_1instance}{tgen\+::sequence\+::instance}} (see \doxylink{group__sequence__inst}{Sequence instances}) from this set of sequences (that satisfies the restrictions) can be generated with {\ttfamily \doxylink{group__sequence__gen_ga0dc45ca71b1cc8087eac28188334dc91}{tgen\+::sequence\+::gen}}.

\begin{DoxyNote}{Note}
Generation might fail if restrictions are considered to be too complex! The restrictions are considered too complex if the following condition does not hold. Consider the graph defined with vertices as the {\ttfamily \doxylink{group__sequence__gen_gaeb0baa4e6c780d8bc1d832514b6fcacd}{tgen\+::sequence\+::distinct}} sets added, and there is an edge between two vertices if they share an index. Note that you need to consider indices (directed or indirectly) connected by equality constraints as being the same. This resulting graph must be a acyclic. Moreover, for every tree in this graph, all of its indices with a {\ttfamily \doxylink{group__sequence__gen_gaff196db1265936038bcb8761192f5a8c}{tgen\+::sequence\+::set}} operation must be covered by a single distinct constraint set.
\end{DoxyNote}
