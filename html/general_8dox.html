Generates a random selection of elements.

Overview

gen_until repeatedly calls a generator until a predicate returns true, or a maximum number of tries is reached.

Properties

  • Stops early if predicate succeeds
  • Deterministic given a deterministic generator
  • Throws on failure (or returns std::optional)

Example

auto inst = gen_until(100, [](const instance& x) {
return x.size() >= 5;
});
